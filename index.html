How to use the Website

Step 1 : Login/Register

Visit the or pages.

Logging in to this website will allow your decks to be saved so that they can be retrieved later.

Step 2 : Vote on Cards

Visit the page.

This is an optional step that will allow to you vote on cards and their effectiveness against other clans.

Do not worry if you are not knowledgeable about the matchups. The L5R Community will vote on the cards.

Step 3 : Build a Deck

Visit the page.

This page will allow users to generate legal decks in order to try new clans or get into the game without too much thought.

Step 4 : Copy the Deck and Play!

Visit the page.

These decks are able to be copied and pasted into the Jigoku website so that you can play them.

You can also view the cards so that you can build them and play with friends or competitions.
